Miniuni is a social network website which focuses social interactions
of students specifically those studying in unversities.

To execute the project:

- Extract the zip file.

- Copy the project folder "Web Engineering Project" into location
C:\wamp64\www\

- Copy the database folder "miniuni" to location C:\wamp64\bin\mysql\mysql5.7.23\data\ 

- Now start your wamp server.

- Open "localhost/web engineering project" in your browser.
